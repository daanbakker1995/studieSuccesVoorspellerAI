## Setup

**requirements**
- Python 3.7.*
- Pip 20.1 or higher

Install the required dependencies from the _requirements.txt_ file with the following command:

``
pip install -r requirements.txt
``

## Running the app

Execute the following command in the root folder to run the Flask API.

``
python app.py
``
